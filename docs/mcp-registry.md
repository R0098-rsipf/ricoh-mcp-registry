# MCP Registry サーバー一覧

このドキュメントでは、MCP Registry に登録されているサーバーの一覧を記載しています。

## 登録サーバー

| 名前 | 説明 |
|------|------|
| [com.github/github-mcp-server](https://github.com/github/github-mcp-server) | リポジトリ、Issue、PR、Actionsなど全GitHub機能を自然言語で操作 |
| [ai.oraios/serena](https://github.com/oraios/serena) | セマンティックなコード取得・編集機能を提供するパワフルなコーディングエージェントツールキット |
| [com.upstash/context7-mcp](https://github.com/upstash/context7) | Up-to-date code docs for any prompt using Context7 |

## 更新履歴

- **2026-01-28**: Context7 MCP を追加、Serena を追加
- **2025-11-28**: GitHub MCP Server を追加（初版）
